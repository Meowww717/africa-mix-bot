# 🌍 Африканці — Telegram Bot

Бот для організації міксових тренувань у групі. Учасники записуються самостійно або парою (чоловік + жінка), адмін керує зустрічами та базою гравців.

---

## 📁 Файли

| Файл              | Призначення                                |
| ----------------- | ------------------------------------------ |
| `bot.py`          | Основний бот (aiogram)                     |
| `import_users.py` | Перегляд учасників групи (telethon)        |
| `bot.db`          | SQLite база даних (зберігається на volume) |

---

## ⚙️ Запуск

### 1. Встановити залежності

```bash
pip install aiogram telethon
```

### 2. Встановити змінні середовища (fly.io secrets)

```bash
fly secrets set TOKEN=ваш_токен_від_botfather
fly secrets set ADMIN_ID=ваш_telegram_id
fly secrets set TELEGRAM_API_ID=ваш_api_id
fly secrets set TELEGRAM_API_HASH=ваш_api_hash
```

### 3. Задеплоїти на fly.io

```bash
fly deploy
```

## 👤 Ролі

| Роль        | Хто                             |
| ----------- | ------------------------------- |
| **Адмін**   | `ADMIN_ID` з env змінної        |
| **Учасник** | Будь-хто хто є в базі `users`   |
| **Гість**   | Людина не в чаті, додана вручну |

---

## 📋 Команди

### Адмін

| Команда         | Опис                                      |
| --------------- | ----------------------------------------- |
| `/create`       | Створити нову зустріч (обрати день і час) |
| `/manage_users` | Відкрити панель управління учасниками     |

> `/create` викликається в груповому чаті — вибір дня і часу приходить адміну в особисті, зустріч публікується в чат.

### Панель `/manage_users`

| Кнопка               | Дія                                                 |
| -------------------- | --------------------------------------------------- |
| ➕ Додати учасника   | Вручну: `user_id, ім'я, прізвище, username, gender` |
| ❌ Видалити учасника | Видалити юзера за `user_id`                         |
| 📋 Показати всіх     | Список учасників з позначкою статі (ч)/(ж)          |

---

## 🏐 Кнопки зустрічі

| Кнопка              | Хто може                    | Дія                                                    |
| ------------------- | --------------------------- | ------------------------------------------------------ |
| 🏐 Записатись       | Учасник з бази              | Сам(а) / З партнером / З гостем                        |
| 👫 Додати партнера  | Записаний одиночно          | Список вільних з бази + вже записані одиночні + вручну |
| 👤 Додати гостя     | Будь-який записаний гравець | Самостійно або парою, з вибором статі                  |
| 🔀 Розбити по парах | Тільки адмін                | Рандомно об'єднує одиночних чоловіків і жінок          |
| ❌ Скасувати запис  | Учасник з бази              | Тільки себе або себе і партнера                        |
| 🗑 Видалити зустріч | Тільки адмін                | Видаляє зустріч і всіх учасників                       |

---

## 📝 Як записатись

1. Натисни 🏐 Записатись
2. Обери варіант:
   - **Сам(а)** — одиночний запис
   - **З партнером з чату** — список людей протилежної статі: вільні з бази, вже записані одиночні, або ввести вручну
   - **З гостем** — вводиш ім'я, партнер автоматично іншої статі
3. Якщо записався одиночно і хочеш додати партнера пізніше — кнопка 👫 Додати партнера

---

## 🔀 Розбивка по парах

Адмін натискає 🔀 Розбити по парах — бот рандомно об'єднує одиночних чоловіків і жінок. Якщо всі одиночні однієї статі — бот повідомляє що пари неможливо створити. Ті хто вже в парі — не чіпаються. Якщо хтось залишається без пари — залишається одиночним.

## 🗄 База даних

### `users`

| Поле         | Опис                |
| ------------ | ------------------- |
| `user_id`    | Telegram ID         |
| `first_name` | Ім'я                |
| `last_name`  | Прізвище            |
| `username`   | @username           |
| `gender`     | `male` або `female` |

### `meetings`

| Поле         | Опис                   |
| ------------ | ---------------------- |
| `meeting_id` | Унікальний ID зустрічі |
| `message_id` | ID повідомлення в чаті |
| `chat_id`    | ID чату                |
| `day`        | День тижня             |
| `time`       | Час                    |

### `participants`

| Поле           | Опис                         |
| -------------- | ---------------------------- |
| `meeting_id`   | ID зустрічі                  |
| `user_id`      | ID юзера (від'ємний = гість) |
| `display_name` | Ім'я для відображення        |
| `pair_id`      | ID пари (NULL = одиночний)   |
| `gender`       | `male` або `female`          |

---

## ⚠️ Важливо

- База даних зберігається на fly.io volume `/data/bot.db` — не зникає після деплою
- При першому деплої створи volume: `fly volumes create bot_data --region fra --size 1 --yes`
- Якщо переходиш зі старої версії — додай колонку: `ALTER TABLE participants ADD COLUMN gender TEXT`
- Всі технічні повідомлення бот надсилає в особисті, груповий чат залишається чистим
