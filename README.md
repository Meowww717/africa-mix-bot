# 🌍🔥 Африканці — Telegram Bot

Бот для організації танцювальних зустрічей у групі. Учасники записуються самостійно або парою, адмін керує зустрічами та базою гравців.

---

## 📁 Файли

| Файл              | Призначення                                 |
| ----------------- | ------------------------------------------- |
| `bot.py`          | Основний бот (aiogram)                      |
| `import_users.py` | Імпорт учасників з групи в базу (telethon)  |
| `bot.db`          | SQLite база даних (створюється автоматично) |

---

## ⚙️ Запуск

### 1. Встановити залежності

```bash
pip install aiogram telethon
```

### 2. Встановити токен бота

```bash
export TOKEN=ваш_токен_від_botfather
```

### 3. Імпортувати учасників з групи (один раз)

```bash
python import_users.py
```

### 4. Запустити бота

```bash
python bot.py
```

> `bot.db` зберігається на диску — дані не зникають після перезапуску бота.

## 📋 Команди

### Адмін

| Команда         | Опис                                      |
| --------------- | ----------------------------------------- |
| `/create`       | Створити нову зустріч (обрати день і час) |
| `/manage_users` | Відкрити панель управління учасниками     |

### Панель `/manage_users`

| Кнопка               | Дія                                                 |
| -------------------- | --------------------------------------------------- |
| ➕ Додати учасника   | Вручну: `user_id, ім'я, прізвище, username, gender` |
| ❌ Видалити учасника | Видалити юзера за `user_id`                         |
| 📋 Показати всіх     | Список учасників з іконкою статі                    |

---

## ⚽ Кнопки зустрічі

| Кнопка              | Хто може                    | Дія                                  |
| ------------------- | --------------------------- | ------------------------------------ |
| ⚽ Записатись       | Учасник з бази              | Бот питає — сам(а) чи з парою        |
| 👤 Додати гостя     | Будь-який записаний гравець | Одного: `Валя`. Пару: `Валя + Льоша` |
| ❌ Скасувати запис  | Учасник з бази              | Видаляє себе і пару зі списку        |
| 🗑 Видалити зустріч | Тільки адмін                | Видаляє зустріч повністю             |

## 🗄 База даних

### `users`

| Поле         | Опис                |
| ------------ | ------------------- |
| `user_id`    | Telegram ID         |
| `first_name` | Ім'я                |
| `last_name`  | Прізвище            |
| `username`   | @username           |
| `gender`     | `male` або `female` |

### `meetings`

| Поле         | Опис                   |
| ------------ | ---------------------- |
| `meeting_id` | Унікальний ID зустрічі |
| `message_id` | ID повідомлення в чаті |
| `chat_id`    | ID чату                |
| `day`        | День тижня             |
| `time`       | Час                    |

### `participants`

| Поле           | Опис                         |
| -------------- | ---------------------------- |
| `meeting_id`   | ID зустрічі                  |
| `user_id`      | ID юзера (від'ємний = гість) |
| `display_name` | Ім'я для відображення        |
| `pair_id`      | ID пари (NULL = одиночний)   |

---

## ⚠️ Важливо

- Запусти `import_users.py` один раз щоб заповнити базу з групи
- `gender` при імпорті ставиться `male` за замовчуванням — зміни вручну через `/manage_users`
- Якщо переходиш зі старої версії бота — видали `bot.db` щоб створилась нова структура таблиць
